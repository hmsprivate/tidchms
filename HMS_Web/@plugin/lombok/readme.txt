Project Lombok makes java a spicier language by adding 'handlers' that know how to build and
compile simple, boilerplate-free, not-quite-java code.

설치방법
* /MSF/pulgin/lombok/lombok.jar 파일을 아래와 같은 방법으로 이클립스에 등록 한다
  * java -jar lombok.jar

사용방법
* http://projectlombok.org/features/index.html 참조
